Power Monitoring statistics

Collect Power Usage information from High Power device that use unconventional power outlets and operating power levels with some sort of Hardware Device. Device will communicate overwifito a database that will store all the information A program will run calculations and create metrics based on the power usage, sending a daily, weekly or month power usage report sorted by section then by device.The program will give suggestions, based on current cost of energy, on devices to move to a different time to save money and energy
